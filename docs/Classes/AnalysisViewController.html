AnalysisViewController

@objc
public final class AnalysisViewController : UIViewController

The AnalysisViewController provides a custom analysis screen which shows the upload and analysis activity. The user should have the option of canceling the process by navigating back to the review screen.

Text resources for this screen

  • ginivision.navigationbar.analysis.back (Screen API only.)

    • Note

    Setting ginivision.navigationbar.analysis.back explicitly to the empty string in your localized strings will make AnalysisViewController revert to the default iOS back button.

    Image resources for this screen

    • navigationAnalysisBack (Screen API only.)

    Resources listed also contain resources for the container view controller. These are marked with Screen API only.
