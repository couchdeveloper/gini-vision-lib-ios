HelpMenuViewController

final public class HelpMenuViewController : UITableViewController

The HelpMenuViewController provides explanations on how to take better pictures, how to use the Open with feature and which formats are supported by the Gini Vision Library.

Text resources for this screen

  • ginivision.navigationbar.help.backToCamera

  • ginivision.navigationbar.help.backToMenu

    • Note

    Setting ginivision.navigationbar.help.backToCamera and ginivision.navigationbar.help.backToMenu explicitly to the empty string in your localized strings will make HelpMenuViewController revert to the default iOS back button.
